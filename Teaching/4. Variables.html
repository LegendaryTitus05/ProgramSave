Hello! In today's lesson, we're going to be learning about Variables!
Variables are things that you can set to a certain number (or letter) and they can also change throughout the program. I'll show you what I mean soon.

First, you write "var".

"var"

Next, you pick a name. It's usually easy to just do a letter, so we'll start with "a". Add a "=", too.

"var a ="

Now, add what you want 'a' to be. (number or letter). We'll do it as a "1".

"var a = 1;"

Okay. Now I'll show you how to use it in a program.


<html>
<script>
var a = 1;
document.write(a + 1");
document.write(a * 1");
document.write(a - 1);
document.write(a / 1);
<script>
<html>

The outcome of this program would be:
2
1
0
1
